# msim-debugger extension

Debugging extension for MSIM simulator in VS Code.

## Building the Extension

To package the extension yourself with a msim-dap binary for your platform, follow these steps:

1. put your desired `msim-dap` binary into the appropriate directory in `bin/`.

2. Run:
```sh
npx @vscode/vsce package
```

3. take the generated `.vsix` file and install it in VS Code using the "Install from VSIX..." option in the Extensions view.
